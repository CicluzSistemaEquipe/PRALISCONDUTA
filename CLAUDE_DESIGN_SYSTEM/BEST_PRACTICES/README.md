# BEST PRACTICES — Referências de classe mundial para a Pralís

Este diretório destila o que 12 empresas de produto fazem de excepcional, traduzido
para o contexto da **Pralís Conduta** — sistema de uma padaria artesanal **quente e
premium**. Cada arquivo extrai *princípios*, não estética.

---

## ⭐ REGRA DE OURO (vale para todos os arquivos)

> **Extrair PRINCÍPIOS — nunca copiar.**
> Aprender UX, produto, motion, arquitetura de informação e performance dessas
> empresas. **NUNCA** copiar layout, identidade, paleta ou componentes.
> Se uma referência levar a algo que **"não parece Pralís"** (frio, genérico,
> infantil, corporativo-sem-alma), **descartar** — a alma de padaria vem primeiro.

A originalidade da Pralís mora exatamente aqui: **nenhuma** dessas empresas é
terrosa, calorosa ou artesanal como a Pralís. Pegamos o rigor de classe mundial
e o fundimos com **alma de forno**.

---

## 🌗 OS DOIS MUNDOS (sempre mapear para os dois)

- 🌙 **App colaborador** — escuro, emocional, engajamento. Jornada, celebração,
  leveza. Dourado de brilho permitido. Modelos: Duolingo, Apple, Shopify.
- ☀️ **Admin / CMS** — claro, produtivo, denso de informação. Tabelas, filtros,
  status, velocidade. **Sem dourado decorativo.** Modelos: Linear, Notion, Stripe.

Superfícies: **dashboard · CMS · app · apresentação (keynote/relatório)**.

---

## 🍞 CANON da Pralís

- Padaria artesanal, **quente e premium**. Calor terroso, não frio digital.
- **Ação laranja**; **dourado** só como brilho (jamais no admin).
- Tipografia **Montserrat**. Guia/mascote: **Lis**.
- App = engajamento · Admin = produtividade.
- Síntese: *princípios de classe mundial + alma de padaria*.
